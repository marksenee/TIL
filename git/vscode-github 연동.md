
## 1. vscode github 설치
-  vscode extension 열기 
- github 입력 -> install 
- vscode 왼쪽 메뉴바에 git 메뉴생성되었다면 성공!
<br/><br/>

## 2. github 계정 연결 
- github 로그인 및 config에 등록해줘야 함 
- 아래의 코드를 터미널에 입력 
```
git config --global user.name "github 사용자 이름"
git confing --global user.email "github 이메일"
git config --list
```
